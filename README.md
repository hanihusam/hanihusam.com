## Developing

A nodejs >= 8.0.0 setup with [yarn](https://yarnpkg.com/) is recommended.

```bash
# install dependencies
yarn

# ...or if you'd like to use npm instead
npm install

# serve with hot reload at localhost:3000
yarn dev

# build for production
yarn build
```
